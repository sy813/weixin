# Flask部署微信公众号后台
【传送门】Ubuntu下部署Flask应用 http://xinyuez.xyz/article/?id=9

##功能介绍：

1. 发送“笑话”，将自动随机回复一个糗事百科文字笑话；
2. 发送“天气”+“地名”将自动回复7天天气预报（中国天气网数据），如“天气北京”；
3. 发送“股票”+“代码”或“名字”将自动回复当日股价。
